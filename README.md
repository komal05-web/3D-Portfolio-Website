# 🚀 3D Portfolio Website

An interactive 3D developer portfolio built with **React**, **Three.js**, and **styled-components**.  
This project showcases my skills, projects, and experience in a visually engaging way using 3D graphics and smooth animations.

---

## ✨ Features

- 🎨 **Modern UI/UX** with responsive design
- 🌐 **3D Models & Animations** powered by Three.js / React Three Fiber
- 🖼️ **Project Showcase** with images, descriptions, and GitHub links
- 📱 **Responsive Layout** for desktop and mobile
- ⚡ **Smooth Transitions** and hover effects
- 🔗 **Social Links** and contact section

---

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite/CRA, styled-components, TailwindCSS (optional)
- **3D Graphics:** Three.js, React Three Fiber, @react-three/drei
- **Animations:** Framer Motion
- **Deployment:** Vercel / Netlify / GitHub Pages
